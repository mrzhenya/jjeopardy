Package net.curre.jjeopardy.images Description

Contains image resources for the JJeopardy application.

To retrieve an image from this location, you can use net.curre.jjeopardy.App as an anchor point and write something like this: new ImageIcon(App.class.getResource("images/info-24.png")) or just use a helper method on the ImageEnum, e.g. ImageEnum.INFO_24.toImageIcon().
