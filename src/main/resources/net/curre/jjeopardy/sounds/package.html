Package net.curre.jjeopardy.sounds Description

Contains sound resources for the JJeopardy application.

To retrieve a sound from this location, you can use net.curre.jjeopardy.App as an anchor point and write something like this: App.class.getResourceAsStream("sounds/boo.wav") or just use a helper method on the SoundEnum, e.g. SoundEnum.BOO_0.getSoundFileStream().
